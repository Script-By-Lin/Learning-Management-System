# Learning Management System (LMS) - Media Upload Specifications

This document outlines the recommended file sizes, dimensions, and aspect ratios for various media upload assets across the Learning Management System.

---

## 📸 Media Sizing Guidelines

### 1. User Profile Picture (Avatar)
* **Recommended Aspect Ratio**: `1:1` (Square)
* **Recommended Dimensions**: `512 x 512` pixels (or `256 x 256` pixels minimum)
* **Supported File Formats**: `.jpg`, `.jpeg`, `.png`, `.webp`
* **Maximum File Size**: `2 MB`
* *Purpose*: Ensures a clean, high-resolution rendering on user profiles, comments, discussion threads, and messages without distortion.

### 2. LMS Web Site Banner
* **Recommended Aspect Ratio**: `16:9` or wider (Panoramic / Aspect ratio of `4:1` works best)
* **Recommended Dimensions**: `1920 x 480` pixels (or `1200 x 300` pixels minimum)
* **Supported File Formats**: `.jpg`, `.jpeg`, `.png`, `.webp`
* **Maximum File Size**: `5 MB`
* *Purpose*: Provides a premium, high-fidelity hero image overlay on desktop and mobile viewports.

### 3. Course Thumbnail
* **Recommended Aspect Ratio**: `16:9` (Standard Widescreen)
* **Recommended Dimensions**: `1280 x 720` pixels (or `640 x 360` pixels minimum)
* **Supported File Formats**: `.jpg`, `.jpeg`, `.png`, `.webp`
* **Maximum File Size**: `3 MB`
* *Purpose*: Displays cleanly in the public course catalog, student dashboards, and enrollment previews.
